## 🌍 C'est quoi GRAVITY-Z ?

GRAVITY-Z est un simulateur de gravité interactif développé dans le cadre d'un projet scolaire.

Il permet de placer des balles dans un environnement 2D via la souris, de configurer leur masse et leur taille, puis d'observer leur chute simulée en temps réel.

À l'impact avec le sol, les statistiques de vol sont affichées : vitesse, durée de chute, trajectoire.

Le tout propulsé par une communication WebSocket entre un frontend web et un serveur C++/Qt.
